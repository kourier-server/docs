Configuring Server

HttpServer provides the setServerOption and serverOption methods that you can use to configure and check the server's configuration.

The ServerOption enum describes the server options that you can configure. Default, minimum, and maximum values for these options are shown in the table below, and you can call errorMessage to get a description of the error if setServerOption returns false.

HttpServer applies the configuration you set when it starts. If you change any configuration for an already running server, HttpServer will only use it the next time it starts.

You can call HttpServer::setTlsConfiguration to configure TLS encryption for HttpServer according to a given TlsConfiguration.
